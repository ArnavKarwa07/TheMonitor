# Monitor Logo Generation Prompts

This file contains several high-quality prompts optimized for different AI image generators (DALL-E 3, Midjourney, Imagen) to create a clean, modern, minimalist logo/app icon for the **Monitor** application.

---

## Prompt 1: High-Tech Vision AI (Recommended)
> **Description:** Ideal for Midjourney (v6) or DALL-E 3. Focuses on the real-time AI skeleton posture overlay.

**Prompt:**
```text
A minimalist, modern vector app icon for a software named "Monitor". The icon features a stylized human spine and neck silhouette transitioning from a curved, unaligned posture in muted orange, to a straight, perfectly aligned vertical line in vibrant emerald green. A clean, subtle camera lens aperture icon is integrated at the top of the vertical spine, representing vision-based posture correction. The design is flat, set within a rounded squircle shape with a deep dark slate-blue (#181825) background. Clean, professional tech aesthetic, high contrast, no text.
```
